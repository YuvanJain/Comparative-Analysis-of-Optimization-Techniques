# Comparative Analysis of Optimization Techniques for PCOS Prediction

## Project Overview
This project presents a comparative analysis of different feature selection and dimensionality reduction techniques‚Äî**PCA**, **Genetic Algorithm (GA)**, **Particle Swarm Optimization (PSO)**, and **Ant Colony Optimization (ACO)**‚Äîapplied to the prediction of **Polycystic Ovary Syndrome (PCOS)** using machine learning models.

The objective is to evaluate how various optimization techniques impact model performance, accuracy, and feature relevance in a healthcare prediction task.

---

## Objectives
- To reduce feature dimensionality and redundancy in the PCOS dataset
- To compare traditional dimensionality reduction (PCA) with metaheuristic optimization techniques
- To analyze the effect of feature selection on model performance
- To identify the most effective optimization method for PCOS prediction
